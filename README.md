# ECommerce

Este projeto foi gerado usando [Angular CLI](https://github.com/angular/angular-cli) version 22.0.4.

## Servidor de desenvolvimento
Para iniciar um servidor de desenvolvimento local, execute:,

```bash
ng serve
```

Assim que o servidor estiver rodando, abra o seu navegador e acesse http://localhost:4200/. A aplicação será recarregada automaticamente sempre que você modificar qualquer um dos arquivos fonte.

## Estrutura básica de código

O Angular CLI inclui ferramentas poderosas de geração de código base. Para gerar um novo componente, execute:

```bash
ng generate component component-name
```

Para uma lista completa de schematics disponíveis (como `components`, ´`directives` ou `pipes`), execute:

```bash
ng generate --help
```

## Compilação

Para compilar o projeto, execute:

```bash
ng build
```

Isso irá compilar o seu projeto e armazenar os artefatos de build no diretório `dist/`. Por padrão, o build de produção otimiza a sua aplicação para desempenho e velocidade.

## Executando testes unitários

Para rodar os testes unitários com o [Vitest](https://vitest.dev/) test runner, use o seguinte comando::

```bash
ng test
```

## Executando testes de ponta a ponta

Para testes de ponta a ponta (e2e), execute:

```bash
ng e2e
```

O Angular CLI não vem com um framework de testes de ponta a ponta por padrão. Você pode escolher um que atenda às suas necessidades.
